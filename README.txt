Speed typing game made using HTML5, CSS3, Bootstrap 4 and Javascript.

Rules are simple, type the displayed word as fast as you can before time runs out.
Initialy you will have 5 seconds to type the designated word, but as you progress through the game the inital time limit will decrease
making the game progressively more challenging.

Words that are displayed are hardcoded and picked randomly. It is possible to excounter the same word twice or more times in a row. 

Hone your typing skills and take the challenge!
